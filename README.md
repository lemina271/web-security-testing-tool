# Web Application Security Testing Tool

This project is a Python-based GUI tool that helps test web applications for common security vulnerabilities such as:

- SQL Injection
- Cross-Site Scripting (XSS)

## Features
- Simple GUI using Tkinter
- Automated payload testing
- Real-time scan results
- Auto-generated security report

## Technologies Used
- Python
- Tkinter
- Requests library

## How to Run
1. Install Python 3
2. Install required library:
   pip install requests
3. Run the tool:
   python gui.py

## Disclaimer
This tool is for educational purposes only.  
Test only on websites you own or have permission to test.
